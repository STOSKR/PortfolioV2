# Proyecto de Ejemplo

Este es un proyecto de ejemplo para demostrar cómo funcionan las Content Collections en Astro.

## Características

- Content Collections con Zod
- TypeScript estricto
- TailwindCSS para estilos
- Código limpio y mantenible

## Instalación

```bash
npm install
npm run dev
```

## Uso

Este es solo un ejemplo para demostrar la estructura del proyecto.
