## Imágenes Hero (Principal)

### IMPORTANTE: Reemplaza estos archivos con tus fotos:

**hero-normal.jpg**
- Tu foto normal que aparece al entrar al sitio
- Esta es la imagen base que siempre está visible

**hero-hover.jpg**  
- Tu foto con casco/efecto especial que aparece al pasar el cursor
- Solo se ve en un círculo de 200px siguiendo el cursor

### Especificaciones técnicas:
- **Formato**: JPG, PNG o WebP
- **Resolución mínima**: 1920x1080px (Full HD)
- **Proporción**: 16:9 horizontal
- **Peso optimizado**: 200-500KB por imagen
- **Compresión**: Usa TinyPNG o similar antes de subir

### Efecto Spotlight:
El efecto hace que solo se vea la imagen hover donde está el cursor (círculo de 200px).
El resto de la pantalla muestra la imagen normal.

### Cómo reemplazar:
1. Prepara tus dos fotos (normal y con casco)
2. Renómbralas a `hero-normal.jpg` y `hero-hover.jpg`
3. Colócalas en `public/images/` reemplazando los placeholders
4. Recarga la página (Ctrl + Shift + R)

**Ejemplo:**
```
public/images/
  ├── logo.svg
  ├── hero-normal.jpg (TU FOTO NORMAL)
  └── hero-hover.jpg (TU FOTO CON CASCO)
```

## Tips para las fotos:
- Asegúrate de que tu cara/cuerpo esté centrado
- Misma pose en ambas fotos para mejor efecto
- Buena iluminación y contraste
- Fondo oscuro o neutro para consistencia con el diseño
